SQL Injection Info

Ovo je primjer SQL umetanja (SQL Injection). Kada je zaštita isključena, korisnik može kao korisničko ime poslati neki upit koji će rezultirati istinom a za lozinku može upisati bilo što. U takvom slučaju, korisnik će dobiti cijeli sadržaj tablice nad kojom je upit rađen. Ukoliko je zaštita uključena, tada napad SQL umetanjem neće uspjeti budući da su podaci parametrizirani.


Neki od primjera istinitosne vrijednosti u SQL-u su:

' OR 1=1;--

'OR 1=1;--'


Ispod je prikazan sadržaj baze:

Username Password Role
MarkoC 123456789 admin
IvanL qwertzuiop user
LukaL asdfghjkl user
ZvonkoG yxcvbnm user
